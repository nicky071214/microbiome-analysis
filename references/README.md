# References

Use this directory to document taxonomy classifiers and reference databases.

Do not commit very large classifier files unless there is a clear reason. A
QIIME 2 classifier can often be regenerated or downloaded from a documented
source.

Record the following for every classifier:

- database name
- database version
- training region
- primer set
- training command or source URL
- download date
- checksum

Expected starter file name:

```text
references/classifier-full-length-16s.qza
```
